# How to run the program:

## Prerequisites

1.  **Java Development Kit (JDK)**
   


2. **Eclipse IDE** (or any Java IDE)  
   - Download from [https://www.eclipse.org/downloads/](https://www.eclipse.org/downloads/).

---

## Steps to Run

1. Download the repository as a ZIP and extract it.
2. Open Eclipse.
3. Import the project:  
   `File → Import... → Existing Projects into Workspace`
4. Select the folder containing the `Chess` package (`com/ewan/chess`) and click Finish.
5. Open `ChessTester.java`.
6. Press the **Run Button** with `ChessTester.java` open.

---

### Notes

- Console-based game, no GUI.  
- Enter moves using the **square's notation, not the piece** (e.g., `g1 f3` to play 1. Nf3).
